Security Headers

Отворете DevTools → Network → Headers, за да видите HTTP отговора.

HeaderСтойностЦел
X-Content-Type-Options nosniff Предотвратява MIME-sniffing
X-Frame-Options DENY Предотвратява clickjacking (iframe embedding)
Strict-Transport-Security max-age=31536000; includeSubDomains Принудително HTTPS (HSTS)
Content-Security-Policy default-src 'self'; style-src 'self' cdn.jsdelivr.net; … Ограничава от къде могат да се зареждат ресурси → предотвратява XSS
Referrer-Policy strict-origin-when-cross-origin Контролира изпращаната Referer информация 

// Конфигурация в SecurityConfig.java:
http.headers(h -> h
    .frameOptions(fo -> fo.deny())
    .contentTypeOptions(ct -> {})
    .httpStrictTransportSecurity(hsts -> hsts
        .includeSubDomains(true).maxAgeInSeconds(31_536_000))
    .contentSecurityPolicy(csp -> csp
        .policyDirectives("default-src 'self'; ..."))
    .referrerPolicy(rp -> rp
        .policy(ReferrerPolicyHeaderWriter.ReferrerPolicy.STRICT_ORIGIN_WHEN_CROSS_ORIGIN))
);
