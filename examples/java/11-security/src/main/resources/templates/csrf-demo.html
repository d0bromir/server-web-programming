CSRF – Cross-Site Request Forgery

Spring Security добавя CSRF токен автоматично когато използвате th:action. Формата по-долу е защитена. Заявка без валиден токен ще получи 403 Forbidden.
Получено: 

<!-- Thymeleaf автоматично добавя: -->
<input type="hidden" name="_csrf" value="...токен..."/>

// В Spring Security 6 CSRF е включена по подразбиране.
// Изключване само за REST API:
http.csrf(csrf -> csrf.ignoringRequestMatchers("/api/**"));
