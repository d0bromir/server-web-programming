Тема 10 – REST API

Всички endpoints връщат JSON. За POST / PUT / DELETE използвайте curl или друг HTTP клиент.

API токен (за защитени endpoints): secret-token-123
Header: X-API-Token: secret-token-123

Демо endpoints

  • GET /api/info – обща информация
  • GET /api/hello – поздрав (незащитен)
  • GET /api/hello?name=Student
  • GET /api/request-info – метаданни на заявката
  • GET /api/old-venues – 301 редирект (versioning демо)
  • GET /api/headers – custom response headers
  • GET /api/status/200 – произволен HTTP статус
  • GET /api/status/418 – 418 I'm a Teapot

Venues CRUD /api/venues (изисква X-API-Token)

  • GET /api/venues – всички заведения
  • GET /api/venues/1 – по ID
  • POST /api/venues – създаване
  • PUT /api/venues/1 – обновяване
  • DELETE /api/venues/1 – изтриване

curl примери

# Незащитен endpoint
curl http://localhost:8080/api/hello?name=Student

# Защитен – списък
curl -H "X-API-Token: secret-token-123" http://localhost:8080/api/venues

# Създаване
curl -X POST http://localhost:8080/api/venues \
     -H "X-API-Token: secret-token-123" \
     -H "Content-Type: application/json" \
     -d '{"name":"Кафе Централ","city":"Sofia","rating":4.5}'

# Обновяване
curl -X PUT http://localhost:8080/api/venues/1 \
     -H "X-API-Token: secret-token-123" \
     -H "Content-Type: application/json" \
     -d '{"name":"Ново Кафе","city":"Varna","rating":4.8}'

# Изтриване
curl -X DELETE -H "X-API-Token: secret-token-123" http://localhost:8080/api/venues/1
