Как работи MVC?
  1. Браузърът изпраща GET /about
  2. Spring насочва заявката към HomeController.about()
  3. Контролерът добавя данни в Model и връща "about"
  4. Thymeleaf рендерира templates/about.html с данните от модела
  5. HTML страницата се изпраща до браузъра
